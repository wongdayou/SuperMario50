README

/**About**/

This is the executable version for Super Mario 50, which is the 4th assignment
for the CS50 introduction to game development online course by Harvard University
on edx.org.

/**How to use**/
Double click on mario.exe and the project will run in LOVE2D


/**How to play**/
From the home screen, press enter to start the game
Use AD to move the character. Spacebar to jump
Collect the floating keys and jump hit the lock boxes to unlock the flag. Touch the
flag to clear the stage
Clearing each stage will spawn a longer stage. The length of the stage will reset 
back to the original when you die
Keep playing and see how far you can go!
